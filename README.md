## datasave(file_name, file_value, file_directory = None, file_type = None)
```
datasave("score", 3, "./scores", "txt") # score.txt >> 3
```

## dataview(file_name)
```
dataview("score.txt") # string type 3
```
### intdataview(file_name)
```
intdataview("score.txt") # int type 3
```
